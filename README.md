# 💰 Финансовый трекер (Financial Tracker)

🌐 **[Ссылка на проект](ССЫЛКА_НА_VERCEL)**

SPA-приложение для учета личных финансов. Позволяет удобно фиксировать доходы и расходы, отслеживать общий баланс и анализировать финансовую статистику с помощью интерактивных графиков.

## 🚀 Особенности (Features)
* **Управление транзакциями:** добавление доходов и расходов с указанием категорий и дат.
* **Умная фильтрация:** раздельный просмотр всех операций, только доходов или только расходов.
* **Интерактивная аналитика:** визуализация структуры финансов с помощью круговой диаграммы (Recharts), которая адаптируется под выбранный фильтр.
* **Локальное хранение:** данные сохраняются на устройстве пользователя и не пропадают после перезагрузки страницы (имитация API через `localStorage`).
* **Модульная архитектура:** использование глобального состояния (Context API) и изолированных бизнес-функций (Custom Hooks).

## 🛠 Технологии (Tech Stack)
* **Frontend:** React 18, TypeScript
* **Стейт-менеджмент:** Context API + `useReducer`
* **Визуализация данных:** Recharts
* **Тестирование:** Vitest + React Testing Library + happy-dom
* **Сборка:** Vite
